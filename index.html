Summary

I write software that takes HTTP requests and spits out HTTP responses. It's my very favorite thing.

Since 2014 I've been doing it at SoFi, where we are eating the lunches of every industry that's based on compounding interest. We use the following technologies:

  • Play! 2.5, a JVM web framework, with routing and templating and CSRFing and databasing and reactive-ing and all that jazz. Play supports Scala, but I use Java.
  • Angular 2, a highly opinionated frontend straightjacket
  • Docker, more specifically docker-compose, which allows us to reliably manage a bunch of containers and abstract a lot of dependencies away for day-to-day development. It's the future, except it actually is the future.
  • Atlassian's stack for all manner of collaboration, CI and general businessing. That means Bitbucket, JIRA, Bamboo, HipChat, and Confluence
